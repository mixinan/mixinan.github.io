使用VS Code编写C语言程序

VS Code 是微软开发的编辑器。打开速度很快，支持的功能也非常丰富——有各种各样的插件。本篇介绍如何用它编写C语言程序。

准备工作

先下载 VS Code、MinGW 等软件。
VS Code 不是IDE（集成开发环境）。它只是一个编辑器，所以编译器要自己安装。
MinGW 的默认安装目录是 C:\MinGW，安装完要配置环境变量：把 bin 目录放入 PATH 中。然后在 cmd 窗口执行 gcc -v 测试。
