本博客的创建流程

Apr 20, 2018 | 教程 | 阅读

表达想法、总结新知识、收录技术问题的解决方案、记录一些繁琐的流程（比如像本篇一样的教程类文章）……写博客是一个非常好的途径。成熟的博客平台有很多，但偶尔还会看到有些写作者拥有自己的博客网站。作为技术人员，我特别好奇他们是怎么做到的。

2016年初我尝试用 阿里云主机 + wordpress 搭建过个人博客，并更新过大半年，写了一些东西。后来接触到云服务器，我很喜欢，开始用它鼓捣各种技术，不惜放弃我的博客。

用GitHub生成个人主页，特别简单，我很早就玩过，所以王晓洋给我推荐搭建博客的方法时，我看到他提到了GitHub，就完全没有在意（虽然我回复他说“厉害厉害”）。前几天突然想起来，又去问他那个框架名字，找了几个帖子把新博客搭了出来。使用很满意。我把搭建流程记录一下。

需要用到的：

技术 描述
markdown 编写博文样式要用的语言
GitHub 博客代码和内容的托管平台

Node.js 服务器语言
hexo 著名的博客框架
